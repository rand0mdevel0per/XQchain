pub fn verifying_key(&self) -> VerifyingKey<P>

This auxiliary function derives a VerifyingKey from a bare SigningKey (even in the absence of the original seed).

This is a utility function that is useful when importing the private key from an external source which does not export the seed and does not provide the precomputed public key associated with the private key itself.

SigningKey implements signature::Keypair: this inherent method is retained for convenience, so it is available for callers even when the signature::Keypair trait is out-of-scope.
